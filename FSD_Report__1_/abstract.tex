 The "Route Optimizer" (RouteWise) platform is an intelligent travel itinerary planning system developed using the MERN stack (MongoDB, Express, React, and Node.js). The application addresses the complex challenge of arranging multiple travel destinations into an efficient sequence, commonly known as the Traveling Salesperson Problem (TSP). By implementing the Nearest Neighbor algorithm based on Haversine distance calculations, the system provides users with optimized routes that minimize travel time and fuel costs.

The platform integrates real-time geospatial data enrichment through the Open-Meteo API for live weather forecasts and the Overpass API (OpenStreetMap) for identifying nearby facilities such as hotels and fuel stations. Featuring a premium, glassmorphic user interface, Route Optimizer offers an immersive experience that allows travelers to manage stops, visualize paths on interactive maps, and retrieve saved itineraries from a personal dashboard. This project demonstrates a robust integration of modern web technologies and optimization algorithms to simplify logistical planning for the modern traveler.
